# About
This is a demo work of mine. I used redux templete to build a page that serve data(redux state) via socket.io. This demo directly use a variable on the server as its database.   
#How to Start
## Installation
```bash
npm install
```
## Running Dev Server

```bash
npm run dev
```
## Look at 
localhost:3000/mypage
